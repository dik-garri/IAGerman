API-ключ Gemini

Приложение работает без сервера, поэтому нужен ваш бесплатный ключ Gemini. Он хранится только на этом устройстве (в браузере) и никуда не отправляется, кроме самого Google.

  1. Откройте aistudio.google.com/app/apikey
  2. Нажмите «Create API key» (бесплатно)
  3. Скопируйте ключ и вставьте сюда

Если на вашем ключе квота модели = 0 (ошибка 429), попробуйте вариант -lite.
